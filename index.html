API Name API Description
weather Returns weather observations from a weather station which is found within the parameters. The parameters used are North, South, East and West, coordinates of bounding box.
Latitiude max and min values of 90 and -90. Longitude man and min 180 and -180.
Example: Berlin will be covered with coordinates N=52.5, S=52.1, W=12.3 & E=14.2
timezone Returns the country, sunrise and sunset times, time and timezone ID of location. The parameters are latitude and longitude coordinates. The numbers are in decimal degrees format and range from -90 to 90 for latitude and -180 to 180 for longitude.
Example: Edinburgh coordinates are latitude = 55.9 and longitude = -3.2
neighbours Returns the number of neighbours and the names of the neighbours of a selected country.
